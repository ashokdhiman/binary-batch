Q1. What is a Programming Language?
Ans: A programming language is a formal language comprising a set of instructions that produce various kinds of output. It is used to communicate with computers to develop software applications, algorithms, and more.

Q2. Why do we need a programming language?
Ans: Programming languages are needed to:
1. Instruct computers to perform specific tasks.
2. Solve problems and implement algorithms.
3. Develop applications, software, and websites.
4. Facilitate communication between the programmer and the machine.

Q3. What are the features of Java?
Ans: Java has the following key features:
1. **Platform Independent**: Code runs on any platform with a Java Virtual Machine (JVM).
2. **Object-Oriented**: Supports concepts like classes, objects, inheritance, etc.
3. **Secure**: Provides a secure runtime environment.
4. **Robust**: Includes strong memory management and exception handling.
5. **Multithreaded**: Allows concurrent execution of threads.
6. **Portable**: Can be carried and executed anywhere without modifications.

Q4. What is an Object?
Ans: An object is an instance of a class. It represents a real-world entity with state (attributes) and behavior (methods). Objects are created using the `new` keyword in Java.

Q5. What is a Class?
Ans: A class is a blueprint or template for creating objects. It defines the state (fields/variables) and behavior (methods/functions) that the objects created from it will have.
